**PROMPT**

Given an `n x n` array, write a function that returns the array elements
arranged from outermost elements to the middle element, traveling clockwise.

Example Collection:
```
  const arrayMatrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
  ];
```

Example Response:
```
  snail(arrayMatrix) #=> [1, 2, 3, 6, 9, 8, 7, 4, 5]
```
